# Getting Started

## Installation

```bash
pip install mkdocs-zen-mode
```

## Add to your project

In your `mkdocs.yml`:

```yaml
plugins:
  - search
  - zen-mode
```

## Usage

Once installed, you'll see a new icon in the header bar. Click it to enter Zen Mode.

### Keyboard Shortcut

Press **Alt+Z** to toggle Zen Mode without clicking.

### Persistent State

Your preference is saved in the browser's localStorage. If you enable Zen Mode, it stays enabled even when you navigate to other pages or close and reopen the browser.
